Lost?

This is the backend/API for Thribhu's Past Paper Repository. You might be lost if you ended up here.

Currently, the backend is being run with the --api-only flag, which means the website is not hosted here.

If you are a viewer, contact the admin of the site (could be school admin or just the website itself), or provide this website as Set Custom API at the bottom of the landing page.

If you are an admin, you may have configured the Python server wrong. Try removing the --api-only flag.

If you are a developer (or curious), take a look at /api/docs to see all endpoints.
